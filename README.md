## Project Description
This is group Beta 4's "Red Raider JobConnect" based on our Figma designs from Project 1. It is a job application for junior and senior level college students to stream line and simplify the job/internship search.
## Pages Implemented
1. Landing Page (index.html)
2. Student Discovery (students.html)
3. Company Discovery (companies.html)
4. Job/Interview Listings (Jobs.html)
